About Me

Perkenalkan Nama saya Guruh Maulana asal universitas STMIK IKMI Cirebon Mahasiswa Semester 3 Jurusan D3 Komputerisasi Akuntansi, ini adalah portofolio pertama saya yang dibuat dengan bootstrap 5 saya belajar pemrograman web ini secara otodidak dari Youtube dan juga kursus pemrograman web dasar front end.

Di jurusan D3 Komputerisasi Akuntansi saya diajarkan bagaimana caranya menyusun laporan keuangan menggunakan siklus akuntansi dasar dari perusahaan jasa dan perusahaan dagang dan juga saya diajarkan cara mengaplikasikan siklus akuntansi menggunakan sofatware akuntansi zahir accounting. dijurusan ini juga akan diajarkan tentang IT dari pemrograman web dasar, pemrograman python dan juga analisis data.
